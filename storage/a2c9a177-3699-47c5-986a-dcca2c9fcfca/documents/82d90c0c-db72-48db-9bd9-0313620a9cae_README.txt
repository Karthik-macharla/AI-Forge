HOW TO BUILD THE EXE
=====================

Requirements:
  - .NET 8 SDK installed (https://dotnet.microsoft.com/download)

Steps:
  1. Open a terminal/command prompt in this folder
  2. Run:  dotnet publish -c Release -r win-x64 --self-contained true -o ./publish
  3. Your SampleApp.exe will appear in the ./publish folder

Or simply double-click build.bat on Windows.

Self-contained = true  -->  EXE runs without .NET installed on target machine
Self-contained = false -->  Smaller EXE, but requires .NET 8 runtime on target machine
